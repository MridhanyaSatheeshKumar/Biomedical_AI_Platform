## System Architecture

This project integrates:

1. Clinical biomarker data (FHIR-based pipeline)
2. Behavioral craving prediction model
3. Rule-based personalized nutrition engine

Pipeline:

Clinical Data → Biomarkers → Risk Prediction
+
Behavior Data → Binge Prediction
↓
Personalized Nutrition Recommendations

## Impact

This system demonstrates how AI can combine clinical and behavioral signals to deliver personalized dietary guidance.
