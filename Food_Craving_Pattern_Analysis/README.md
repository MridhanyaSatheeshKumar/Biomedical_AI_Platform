# Food Craving Pattern Analysis

## Objective
Analyze behavioral triggers behind food cravings.

## Dataset
Synthetic + (future: MyFitnessPal / survey)

## Methods
- Feature engineering (mood, time, trigger)
- Classification model (Random Forest)

## Key Insights
- Stress → high-calorie food
- Night → increased cravings

## Future Work
- Real user data
- Integration with biomarker risk model
